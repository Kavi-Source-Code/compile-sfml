# compile-sfml
Simple script to compile an sfml program on macOS. It definitely works!

To run it:

`cd Desktop`

`git clone https://github.com/Kavi-Source-Code/compile-sfml`

`cd compile-sfml`

`chmod 755 compile-sfml.sh`

`./compile-sfml.sh`

OR:

	bash <(curl -fsSL https://raw.githubusercontent.com/Kavi-Source-Code/compile-sfml/HEAD/compile-sfml.sh)
